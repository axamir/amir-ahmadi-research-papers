# Beyond Models: Toward Enduring Human–AI Collaborative Systems

## A Research Framework for the Future of Human–AI Collaboration

Author: Amir Ahmadi

Status: Research Release v1.0.0

This directory contains the Human–AI Collaborative Systems (HACS) framework.

Core documents:

- `paper.en.md` — English research manuscript
- `paper.fa.md` — Persian research manuscript
- `framework/` — architecture and evaluation documents
- `references/` — bibliography

The framework explores how humans and AI systems can collaborate through capability, context continuity, governance, and responsible interaction.
